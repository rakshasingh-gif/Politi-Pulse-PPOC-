# PolitiPulse: Public Policy Sentiment Analysis

## Overview

**PolitiPulse** is an NLP-based analytics platform developed under the **Public Policy and Opinion Cell (PPOC), IIT Kanpur** to analyze and quantify public sentiment toward newly implemented government policies. The project leverages Natural Language Processing (NLP), sentiment analysis, and interactive visualization to generate actionable insights that support evidence-based policymaking.

As a **Project Mentee**, I contributed to the development of the NLP pipeline, sentiment analysis framework, interactive dashboard, and policy analytics modules while collaborating with mentors and team members throughout the project lifecycle.

## Objectives

* Analyze large-scale public opinion on government policies.
* Quantify policy reception using sentiment analysis techniques.
* Develop an interactive dashboard for real-time policy analytics.
* Generate structured insights to support data-driven policymaking.

## Key Features

* **NLP-based sentiment analysis** for analyzing public opinion.
* **Rule-based sentiment scoring** using the **VADER** framework.
* **Interactive Streamlit Dashboard** for real-time visualization.
* **Policy Approval Index** for measuring public policy reception.
* **Automated data pipeline** for collecting, preprocessing, and analyzing public opinion data.
* **Clause-level sentiment mapping** to identify policy strengths and weaknesses.
* **Recommendation reports** based on sentiment trends and policy analytics.

## Project Workflow

### 1. Data Collection
- Gathered public opinion data from multiple digital platforms.

### 2. Data Preprocessing
- Text cleaning
- Tokenization
- Stop-word removal
- Text normalization

### 3. Sentiment Analysis
- Implemented **VADER Sentiment Analyzer**
- Classified sentiments into Positive, Neutral, and Negative categories

### 4. Policy Analytics
- Generated **Policy Approval Index**
- Mapped sentiment trends to individual policy clauses
- Produced data-driven recommendation reports

### 5. Visualization
- Built an interactive **Streamlit Dashboard**
- Displayed real-time policy sentiment analytics
- Visualized approval trends and key insights
